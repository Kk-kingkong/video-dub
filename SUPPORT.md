# Support

## Before reporting a problem

1. Confirm the version shown in the extension popup.
2. Refresh the active YouTube tab after reloading or updating the extension.
3. Open the install guide and check that the Engine version and protocol match the extension.
4. Remove API keys, cookies, video captions, generated audio, local paths, and other private data from screenshots and logs.

## Where to report

- Use the repository's issue tracker for reproducible bugs and feature requests.
- Use GitHub Discussions, when enabled, for setup questions.
- Use the private process in `SECURITY.md` for vulnerabilities or reports containing sensitive details.

Include the extension version, operating system, whether the optional Engine is installed, concise reproduction steps, and the visible error category. Do not publish API keys or YouTube cookies.

LocalTube Dub is community-maintained software. Support and response times are not guaranteed.
